## cfMeDIP-seq_Data_Resource_Codes

## About
This repository is a collection scripts for the manuscript of "A pan-cancer compendium of 1,294 plasma cell-free DNA methylomes and fragmentomes". The structure of the repository is as follows:

  - R_scripts: These are R scripts used to generate primary results and figures in Results
  - Shell_Scripts: These are ordered scripts for methylomes and fragmentomes analyses.

The related data or data links are provided in Supplementary Table 14 of the article.
